makeDirect3D

fun makeDirect3D(width: Int, height: Int, resourcePtr: NativePointer, format: Int, sampleCnt: Int, levelCnt: Int): BackendTexture

Creates BackendTexture from a Direct3D 12 resource.

The resource is RETAINED by the backend texture; adopting it into an Image (Image.adoptTextureFrom) hands that retain to Skia, which releases it when the image is freed — the caller keeps (and remains responsible for) its own reference.

The resource must be in D3D12_RESOURCE_STATE_COMMON at handoff (the state Skia is told to assume); a caller that transitions it later must notify Skia via the GrBackendTextures::SetD3DResourceState path before Skia samples again.

Parameters

width

  • width of the BackendTexture to be created

height

  • height of the BackendTexture to be created

resourcePtr

  • pointer to an ID3D12Resource; must be not zero

format

  • the DXGI_FORMAT of the resource

sampleCnt

  • sample count of the resource (1 for non-MSAA)

levelCnt

  • mip level count of the resource (1 for no mips)
